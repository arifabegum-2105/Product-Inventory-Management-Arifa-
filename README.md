# Product Inventory Management System

# What is this project?
A web application to manage product inventory. 
It allows users to add, update, delete and view products in a store.

# Technologies Used
- JavaScript
- Node.js
- HTML & CSS

# What I Learned
- How to manage data using CRUD operations
- How to build a backend system
- How to work with databases
